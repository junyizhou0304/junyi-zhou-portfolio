# Personal Portfolio Website

A clean, responsive portfolio website template designed for job seekers.

## What’s included

- `index.html` — single-page portfolio layout
- `styles.css` — responsive styling with modern visual hierarchy
- `script.js` — mobile nav toggle and smooth behavior support

## Customize

1. Replace `Your Name`, `Full-Stack Engineer`, and the project examples in `index.html`.
2. Update the email address and LinkedIn URL in the contact section.
3. Add your own project details, skills, and any additional sections.

## Run locally

Open `index.html` directly in the browser, or use a simple static server:

```bash
cd /Users/junyizhou/portfolio
python3 -m http.server 8000
```

Then open `http://localhost:8000`.
